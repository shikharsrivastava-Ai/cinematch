# Cinematch 🎬

## Description
Cinematch is a movie recommendation system that suggests movies based on user preferences using a dataset.

## Features
- Movie search functionality
- Recommends similar movies
- Simple web interface

## Tech Stack
- Python
- Flask
- HTML/CSS
- Pandas

## Dataset
- TMDB 5000 Movies Dataset

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open in browser:
   http://127.0.0.1:5000/
